# AndroidCore Docusaurus Website

This website renders documentation from `../docs`.

## Commands

```powershell
npm --prefix .\website install
npm --prefix .\website run start
npm --prefix .\website run build
```

## Notes

- Source docs stay in `docs/`.
- Bundled Android asset copies stay in `app/src/main/assets/agents/` and should be kept in sync for guides/overview/architecture/examples.
